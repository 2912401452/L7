## 使用
```javascript
import { PolygonLayer } from '@antv/l7'
const layer = new PolygonLayer();
```
### shape
绘制填充图，shape为```fill``` 常量不支持数据映射

```javascript
layer.shape('fill');
```

### size
填充图无size不需要设置size
### color
同 [layer#color](../layer/#color)

### style
同 [layer#style](../layer/#style)
